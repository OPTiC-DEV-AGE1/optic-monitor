# OPTiC Account Monitor

Account monitoring dashboard for Evony Age I bot accounts.

## Requirements
- Windows 10/11
- NeatBot accounts running the autorun script

## Installation
1. Download `OPTiCAccountMonitor_Setup.exe` from [Releases](https://github.com/JWisniewski16/optic-monitor/releases/latest)
2. Run as Administrator
3. Enter your license key on first launch
4. Add the autorun script to your bot accounts (see Scripts tab in app)

## Updates
The app checks for updates automatically on launch.
